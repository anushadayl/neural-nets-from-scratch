# Neural Networks from Scratch

Implementations built while working through Andrej Karpathy's 
"Neural Networks: Zero to Hero" series, focused on understanding 
the internals of backpropagation and language models from first principles.

## Contents

- **micrograd/** — A minimal autograd engine implementing backpropagation 
  from scratch, including a small neural net library built on top of it.
- **bigram/** — A character-level bigram language model for next-character 
  prediction, exploring statistical and neural approaches to language modeling.

## Why this exists

Built as part of a deep learning foundations track for a research internship 
(DRDO ISSA), as preparation before taking on a larger ML research project.

## Tech

- Python, PyTorch
- Jupyter Notebooks
